部署到 GitHub Pages（你只需要上传/替换这些文件）
=========================================

把本压缩包解压后，直接把以下文件/文件夹提交到你的 GitHub Pages 仓库根目录：
- index.html
- manifest.webmanifest
- sw.js
- icons/  (整个文件夹)

提交后等待 1~2 分钟，打开：
https://www.liruna.icu

在 Chrome/Edge 地址栏会出现“安装”按钮；手机 Chrome 也可“添加到主屏幕”。

更新提示：
如果你以后想强制用户刷新缓存，把 sw.js 里的 CACHE_NAME 改成 v2/v3... 再提交。
